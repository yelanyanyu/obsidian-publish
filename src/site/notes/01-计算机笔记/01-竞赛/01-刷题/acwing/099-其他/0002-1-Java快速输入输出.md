输入两个整数，求这两个整数的和是多少。

#### 输入格式

输入两个整数 $A, B$，用空格隔开

#### 输出格式

输出一个整数，表示这两个数的和

#### 数据范围

$0 \le A, B \le 10 ^ 8$

#### 输入样例

```

```

#### 输出样例
