全称为 Random Access Memory，即支持随机访问的内存，断点后数据丢失。

有两种 RAM：
 - [[02-pages/DRAM\|DRAM]] 全称 Dynamic Random Access Memory，即动态 RAM；
 - [[02-pages/SRAM-静态RAM\|SRAM]] Static Random Access Memory，即静态 RAM；
