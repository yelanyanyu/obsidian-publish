在大端存储中，多字节数据的**高字节**（Most Significant Byte, MSB）存储在**低地址**，而**低字节**（Least Significant Byte, LSB）存储在**高地址**。这种存储方式类似于我们书写数字的顺序，从左到右依次表示数值的高低位。

**示例：**

假设有一个32位的十六进制数 `0x12345678`，在大端存储中的内存布局如下：

| 地址   | 内容     |
| ---- | ------ |
| 0x00 | `0x12` |
| 0x01 | `0x34` |
| 0x02 | `0x56` |
| 0x03 | `0x78` |
