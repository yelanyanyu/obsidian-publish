# 定义
![image.png|800](https://yelanyanyu-img-bed.oss-cn-hangzhou.aliyuncs.com/img/blog/2024/06/20240614150914.png)
```ad-note
title: other
- 当相关系数大于 0 时，称为正相关，相关系数小于 0 时，称为负相关。
- 当相关系数 = 1 时，称为 X 和 Y 是线性相关，就 Y = aX + b。
```
