# 不同根式
如果同时出现 $\displaystyle \sqrt[ n ]{ ax+b },\sqrt[ m ]{ ax+b }$，那么就令 $\displaystyle \sqrt[ l ]{ ax + b }$，其中 $\displaystyle l$ 为 $\displaystyle m,n$ 的最小公倍数。
