我们要做的事情就是给定一个逻辑地址，将这个逻辑与页表项的页号与块号进行匹配。当确定了块号后，我们再确定一个偏移量（offset），来确定数据在块中的位置即可。故而有：
```
逻辑地址A对应的物理地址 = P 号页面在内存中的起始地址+页面偏移量W
页号 = 逻辑地址 / 页面长度 （取除法的整数部分）
页内偏移量 = 逻辑地址 % 页面长度
```
