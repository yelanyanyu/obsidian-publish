## 拆项
例如对于积分：$\displaystyle \int e^{ - \frac{x}{2} } \frac{\cos x-\sin x}{\sqrt[  ]{ \sin x }} \, dx$ 就可以拆成：
$$
\int e^{ - \frac{x}{2} } \frac{\cos x}{\sqrt[  ]{ \sin x }} \, dx - \int e^{ -\frac{x}{2} } \frac{\sin x}{\sqrt[  ]{ \sin x }} \, dx 
$$
