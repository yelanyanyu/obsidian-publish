在小端存储中，多字节数据的**低字节**（LSB）存储在**低地址**，而**高字节**（MSB）存储在**高地址**。这种存储方式与人们从右向左书写数字的顺序类似。

**示例：**

同样是32位的十六进制数 `0x12345678`，在小端存储中的内存布局如下：

| 地址   | 内容     |
| ---- | ------ |
| 0x00 | `0x78` |
| 0x01 | `0x56` |
| 0x02 | `0x34` |
| 0x03 | `0x12` |
